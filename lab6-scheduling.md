# Lab 6: CPU Scheduling Algorithms
### Aim
Understand and simulate classic CPU scheduling algorithms including FCFS, SJF and Round Robin.

### Summary
CPU scheduling manages the order of process execution in operating systems.
- FCFS: First come first served.
- SJF: Shortest job first to minimise waiting time.
- Round Robin: Time slice based fair scheduling for multitasking.

Simulation code is written in Python to calculate waiting time and turnaround time for comparison.
