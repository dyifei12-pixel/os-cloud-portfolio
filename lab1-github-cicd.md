# Lab 1 & 2: GitHub and CI/CD
### Aim
This laboratory task aims to create a GitHub repository, configure GitHub Pages, and set up automatic deployment using GitHub Actions CI/CD workflow.

### Procedure
1. Created a new public GitHub repository named os-cloud-portfolio with a README file.
2. Cloned the repository to local computer using Git command.
3. Enabled GitHub Pages and set the source to the main branch.
4. Created a GitHub Actions workflow file to automatically deploy the website whenever new code is pushed.
5. Committed and pushed all files to remote repository.

### Outcome
The portfolio website is successfully published on GitHub Pages. The CI/CD pipeline works automatically and updates the website after every push.
