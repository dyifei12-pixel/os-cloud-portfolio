# Lab 5: Development inside Virtual Machine
### Aim
Develop and run a simple program inside the Ubuntu virtual machine.

### Procedure
1. Entered Ubuntu terminal via SSH.
2. Created a simple Python program and executed it.
3. Launched a local HTTP server using Python built-in module.
4. Opened the server page from host browser to verify connection.

### Outcome
Simple program runs correctly inside VM. The local web server is accessible from the host system.

Screenshots of Python program running and local web server access are stored in the **images** folder.
