# Lab 7: Memory Management and MMU
### Aim
Understand virtual memory, physical memory and MMU address translation.

### Summary
The Memory Management Unit (MMU) translates virtual addresses into physical addresses.
Paging and segmentation allow efficient memory usage, process isolation and reduced fragmentation. Page replacement algorithms such as FIFO and LRU are used when memory frames are full.
