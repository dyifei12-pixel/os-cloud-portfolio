# Lab 3: HTML, CSS and JavaScript
### Aim
To design and build a static portfolio website using pure HTML, CSS and JavaScript with navigation and basic interactive features.

### Procedure
1. Created index.html for page structure and navigation menu.
2. Designed layout, colour scheme and responsive style using external CSS file.
3. Added JavaScript for smooth scrolling and basic page interaction.
4. Organised links to all laboratory tasks inside the navigation bar.

### Outcome
The website has clear navigation, clean layout and simple interactive functions. It is fully readable on both desktop and mobile devices.
