# Lab 4: Virtual Machine Remote Access
### Aim
Install Ubuntu virtual machine, configure network and enable SSH for remote command-line access.

### Procedure
1. Created an Ubuntu 22.04 virtual machine using VirtualBox.
2. Updated system packages and installed OpenSSH server.
3. Enabled SSH service and allowed SSH through firewall.
4. Connected remotely from host machine using SSH command.

### Outcome
Virtual machine runs normally, SSH connection is stable, and remote terminal access is successful.
